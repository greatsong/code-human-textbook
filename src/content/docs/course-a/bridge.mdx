# 코스 B로의 다리

코스 A 15차시를 모두 마쳤다면, 이제 스스로 프로젝트를 설계하고 실행할 준비가 되었습니다.

## 코스 A에서 배운 것

- 센서로 데이터를 수집하는 방법
- 코드로 자동화하고 기록하는 방법
- 데이터를 다듬고 시각화하는 방법
- AI와 협력하여 패턴을 찾는 방법
- 팀 프로젝트를 수행하고 발표하는 방법

## 코스 B에서 달라지는 것

코스 B는 **자기 주도 프로젝트** 코스입니다. 교사가 정해주는 활동 대신, 여러분이 직접 질문을 세우고 센서를 선택하고 프로젝트를 완성합니다.

레벨별로 난이도와 자율성이 높아집니다.
