# 코스 B: 자기 주도 프로젝트

코스 B는 코스 A를 마친 학생이 스스로 도전하는 프로젝트 코스입니다. 레벨별로 난이도와 자율성이 높아집니다.

## 레벨 구성

| 레벨 | 이름 | 내용 | 시간 |
|------|------|------|------|
| Lv.1 | 탐험가 | 단일 센서 + 단순 출력 | 3시간 |
| Lv.2 | 발명가 | 복합 센서 + 조건 로직 | 3시간 |
| Lv.3 | 연구자 | 자유 선택 + 가설-측정-검증 | 3시간 |
| Lv.4+ | 마스터 | 다중 센서 + AI 통합 | 3시간 |
