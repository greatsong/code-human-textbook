## 미션

센서 **2개 이상**을 조합하여, 여러 데이터를 종합 판단하는 **스마트 시스템**을 만드세요. 바이브 코딩(AI 코드 생성)을 적극 활용하세요.

### 완료 조건
- [ ] 센서 2개 이상에서 데이터를 동시에 읽을 수 있다
- [ ] OLED 대시보드에 여러 센서 데이터를 종합 표시한다
- [ ] 복합 조건(`if/elif/else` + `and/or`)으로 스마트 판단을 한다
- [ ] 바이브 코딩을 활용한 부분이 있다 (프롬프트 기록 포함)
- [ ] 한 줄 설명을 작성할 수 있다

### 프로젝트 1: 스마트 환기 시스템

**센서:** DHT20 온습도 (I2C) + SGP30 공기질 (I2C)

**스마트 판단 로직:**
```
CO2 > 1000ppm AND 온도 > 25°C → "즉시 환기!"
CO2 > 800ppm OR 습도 > 70% → "환기 권장"
그 외 → "쾌적"
```

**OLED 대시보드 설계:**
```
+---------------------------+
| Temp: 24.5C  Humid: 58%   |
| CO2: 823ppm  TVOC: 45ppb  |
| ████████░░░ Comfort: OK   |
| -> 환기 권장               |
+---------------------------+
```

**바이브 코딩 활용 포인트:**
- OLED 레이아웃 코드를 AI에게 요청하세요
- 프롬프트 예시: "Pico WH에서 DHT20과 SGP30 센서를 동시에 읽어서, SSD1306 OLED에 환기 대시보드를 표시하는 MicroPython 코드를 작성해줘"

**확장 아이디어:**
- 환기 시작/종료 시 CO2 변화를 기록하여 환기 효과 측정
- LED로 환기 상태를 색상으로 표시

{/* end */}

### 프로젝트 2: 간이 보안 시스템

**센서:** 초음파 거리 센서 (Digital) + 소리 센서 (ADC)

**스마트 판단 로직:**
```
거리 < 30cm AND 소음 > 60% → "침입 감지! 경보!"
거리 < 30cm → "물체 접근"
소음 > 80% → "큰 소리 감지"
그 외 → "안전"
```

**OLED 대시보드 설계:**
```
+---------------------------+
| Dist: 45.2cm  Noise: 23%  |
| ████░░░░░░ Security: SAFE |
| Alert count: 3            |
| Last: 14:32               |
+---------------------------+
```

**바이브 코딩 활용 포인트:**
- 초음파 센서 + 소리 센서 동시 읽기 코드를 AI에게 요청하세요
- 경보 이벤트를 CSV 로그로 저장하는 코드도 요청할 수 있어요

**확장 아이디어:**
- 경보 발생 시 부저(buzzer) 울리기
- 경보 이벤트 이력을 시간과 함께 기록

{/* end */}

### 프로젝트 3: 종합 쾌적도 대시보드

**센서:** DHT20 온습도 (I2C) + 빛 센서 (ADC) + 소리 센서 (ADC)

**스마트 판단 로직:**
```python
# 각 항목별 점수 (0~100)
temp_score = 100 if 20 <= temp <= 26 else 50 if 18 <= temp <= 28 else 0
light_score = 100 if 30 <= brightness <= 70 else 50
noise_score = 100 if noise < 30 else 50 if noise < 60 else 0

# 종합 쾌적도
comfort = (temp_score + light_score + noise_score) / 3
```

**OLED 대시보드 설계:**
```
+---------------------------+
| T:24C H:55% L:62% N:25%  |
| Temp  ████████░░ 85pt     |
| Light ██████░░░░ 65pt     |
| Noise █████████░ 95pt     |
| Total: 82pt [GOOD]        |
+---------------------------+
```

**바이브 코딩 활용 포인트:**
- 3개 센서 동시 읽기 + OLED 4행 대시보드를 AI에게 요청하세요
- 쾌적도 점수 계산 로직도 AI와 함께 설계할 수 있어요

**확장 아이디어:**
- 시간대별 쾌적도 변화를 CSV로 기록
- 쾌적도가 낮아지는 패턴 분석

## 시간 배분 가이드

| 시간 | 활동 | 팁 |
|------|------|-----|
| 0:00~0:20 | 프로젝트 선택 + 센서 2개 연결 | 센서 포트 충돌에 주의하세요 |
| 0:20~0:50 | 바이브 코딩으로 기본 코드 생성 | AI 프롬프트를 구체적으로 작성하세요 |
| 0:50~1:30 | AI 코드 수정 + 디버깅 | 센서 주소, 포트 확인이 핵심이에요 |
| 1:30~2:00 | 복합 조건 로직 추가 | and/or 조건으로 스마트 판단을 구현하세요 |
| 2:00~2:20 | OLED 대시보드 개선 | 레이아웃을 보기 좋게 다듬으세요 |
| 2:20~2:40 | 테스트 + CSV 저장 확인 | 실제 상황에서 테스트하세요 |
| 2:40~3:00 | 산출물 정리 + 발표 준비 | 바이브 코딩 프롬프트도 기록하세요 |
