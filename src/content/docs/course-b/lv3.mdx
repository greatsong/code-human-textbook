## 미션

스스로 **연구 질문과 가설**을 세우고, 센서 데이터를 **장기간 수집**하여, **AI/데이터 분석**으로 가설을 검증하세요.

### 완료 조건
- [ ] 측정 가능한 연구 질문과 가설을 세웠다
- [ ] 센서 데이터를 30분 이상 자동 수집했다 (CSV)
- [ ] OLED 또는 PC 대시보드로 실시간 모니터링했다
- [ ] 수집한 데이터를 matplotlib 등으로 시각화했다
- [ ] 가설 검증 결과를 한 문단으로 작성했다

### 프로젝트 1: 식물 성장 환경 분석

**연구 질문:** "교실 안 위치별로 빛의 양이 얼마나 다를까? 식물을 어디에 두면 좋을까?"

**가설:** "창가의 평균 조도는 교실 안쪽보다 3배 이상 높을 것이다."

**센서:** 빛 센서 (ADC) -- 2곳에 번갈아 측정 또는 2개 사용

**실험 설계:**
| 항목 | 내용 |
|------|------|
| 독립 변인 | 측정 위치 (창가 vs 안쪽) |
| 종속 변인 | 조도 (lux 추정값) |
| 통제 변인 | 측정 시간대, 날씨, 커튼 상태 |
| 측정 기간 | 1시간 (위치별 30분) |
| 측정 간격 | 10초 |

**대시보드:** OLED에 현재 조도 + 위치 라벨 + 평균값 표시

**분석 방법:**
- matplotlib로 두 위치의 시계열 그래프를 겹쳐서 비교
- 위치별 평균, 최대, 최소값 계산
- 가설 검증: 창가 평균 / 안쪽 평균 = 몇 배?

{/* end */}

  </TabItem>

### 프로젝트 2: CO2 농도와 수업 집중도

**연구 질문:** "수업 시간이 지나면서 CO2 농도가 얼마나 올라가고, 환기하면 얼마나 빨리 떨어질까?"

**가설:** "30명 교실에서 환기 없이 40분이 지나면 CO2가 1500ppm을 넘을 것이다."

**센서:** SGP30 공기질 센서 (I2C) + DHT20 온습도 (I2C)

**실험 설계:**
| 항목 | 내용 |
|------|------|
| 독립 변인 | 시간 (수업 시작부터 종료까지) |
| 종속 변인 | CO2 농도 (ppm) |
| 통제 변인 | 학생 수, 교실 크기, 문/창문 상태 |
| 측정 기간 | 수업 1교시 (40분) + 쉬는 시간 (10분) |
| 측정 간격 | 30초 |

**대시보드:** OLED에 CO2 값 + 경과 시간 + 상태 (쾌적/주의/위험)

**분석 방법:**
- 시간에 따른 CO2 변화 그래프
- 환기 전후 CO2 감소 속도 계산
- 1000ppm(주의 기준)을 넘는 시점 찾기

### 프로젝트 3: 운동 강도와 가속도 패턴

**연구 질문:** "걷기, 달리기, 제자리 뛰기의 가속도 패턴은 어떻게 다를까?"

**가설:** "달리기의 가속도 진폭은 걷기보다 2배 이상 클 것이다."

**센서:** ADXL345 가속도 센서 (I2C)

**실험 설계:**
| 항목 | 내용 |
|------|------|
| 독립 변인 | 운동 종류 (걷기/달리기/제자리 뛰기) |
| 종속 변인 | 3축 가속도 진폭 |
| 통제 변인 | 센서 부착 위치, 측정 시간 |
| 측정 기간 | 동작별 30초 x 3회 |
| 측정 간격 | 0.05초 (20Hz) |

**대시보드:** OLED에 실시간 X/Y/Z 값 + 현재 동작 라벨

**분석 방법:**
- 동작별 가속도 시계열 그래프 비교
- 진폭(최대-최소) 계산
- 패턴 차이를 시각적으로 확인

## 연구 프로젝트 진행 가이드

### Phase 1: 연구 설계 (30분)

```python
# 연구 계획서 템플릿
research_plan = {
    "question": "여러분의 연구 질문",
    "hypothesis": "여러분의 가설",
    "sensor": "사용할 센서",
    "variables": {
        "independent": "독립 변인",
        "dependent": "종속 변인",
        "controlled": ["통제 변인 1", "통제 변인 2"]
    },
    "measurement": {
        "duration_min": 30,
        "interval_sec": 10,
        "total_readings": 180
    }
}
```

### Phase 2: 데이터 수집 (1시간)

- 센서 연결 + 데이터 수집 코드 작성
- OLED 대시보드로 실시간 모니터링
- CSV 파일에 자동 저장
- 바이브 코딩으로 수집 코드를 빠르게 만들어도 좋아요

### Phase 3: 데이터 분석 (1시간)

- PC Python + matplotlib로 그래프 그리기
- 평균, 최대, 최소, 범위 등 기초 통계 계산
- 가설과 실제 데이터 비교

### Phase 4: 결론 작성 (30분)

**가설 검증 보고서 양식:**

```
[연구 질문]
___________________________

[가설]
___________________________

[데이터 요약]
- 총 측정: ___회
- 주요 결과: ___________________________

[결론]
가설이 (지지되었다 / 기각되었다).
왜냐하면 ___________________________

[한계점]
___________________________

[다음 연구 제안]
___________________________
```
