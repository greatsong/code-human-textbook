센서를 꽂고, AI에게 코드를 시키고, 대시보드로 세상을 읽습니다.

Grove 커넥터 하나면 센서가 데이터를 쏟아냅니다.
처음에는 직접 코드를 쓰고, 복잡해지면 AI에게 코드를 맡기고,
모인 데이터는 실시간 대시보드로 시각화하고,
마지막에는 머신러닝으로 패턴을 찾아냅니다.

이 교재를 마치면, **센서 데이터를 수집하고 → AI로 코딩하고 → 대시보드로 보여주고 → ML로 분석하는** 전체 파이프라인을 스스로 설계할 수 있게 됩니다.

---

## 두 가지 코스, 하나의 목표

같은 장비, 같은 센서를 사용하지만 배우는 방식이 다릅니다.
자신에게 맞는 코스를 골라 시작하세요.

<CardGrid>
  <Card title="코스 A: 15차시 수업" icon="open-book">
    **매주 50분, 3막 구성으로 단계별 성장**

    - **1막 — 센서와 친해지기** (4차시): 센서를 연결하고, 코드를 직접 쓰고, 첫 대시보드를 만듭니다
    - **2막 — 바이브 코딩으로 확장** (6차시): AI에게 코드를 시키고, 환경 모니터링 대시보드를 만들고, 복합 시스템을 구축합니다
    - **3막 — 대시보드와 머신러닝** (5차시): 실시간 대시보드를 제작하고, 센서 데이터로 머신러닝에 입문합니다

    직접 코딩 → 바이브 코딩 → 대시보드 → ML까지, 15주에 걸쳐 자연스럽게 확장합니다.

    [코스 A 첫 수업으로 이동하기 →](/ai-physical-computing/course-a/act1/lesson-01/)
  </Card>

  <Card title="코스 B: 프로젝트 4레벨" icon="rocket">
    **3시간씩, 레벨업하며 만들기**

    프로젝트를 직접 설계하고 완성하며 실력을 키우는 코스입니다.

    - **Lv.1 탐험가**: 센서 하나로 작동하는 대시보드를 완성합니다
    - **Lv.2 발명가**: 센서 여러 개를 조합해 스마트 시스템을 만듭니다
    - **Lv.3 연구자**: 바이브 코딩으로 장기 데이터 수집 + AI 분석을 합니다
    - **Lv.4+ 마스터**: 머신러닝 모델을 학습시키고 실제 문제를 해결합니다

    매번 더 높은 도전, 더 큰 대시보드, 더 깊은 AI 활용.

    [코스 B 소개 페이지로 이동하기 →](/ai-physical-computing/course-b/intro/)
  </Card>
</CardGrid>

## 이 교재의 특징

<CardGrid stagger>
  <Card title="납땜 없는 하드웨어" icon="puzzle">
    Grove 4핀 커넥터로 꽂기만 하면 연결 완료.
    브레드보드도, 점퍼 와이어도, 인두기도 필요 없습니다.
    커넥터에 방향이 정해져 있어서 잘못 꽂을 수도 없습니다.
  </Card>

  <Card title="바이브 코딩" icon="star">
    처음에는 직접 코드를 쓰며 원리를 배우고,
    복잡한 코드는 **AI에게 자연어로 설명하면 코드가 나옵니다**.
    "CO2 농도가 1000ppm 넘으면 LED를 빨간색으로 바꿔줘" — 이렇게요.
    AI가 쓴 코드를 읽고, 수정하고, 이해하는 것이 진짜 실력입니다.
  </Card>

  <Card title="실시간 대시보드" icon="laptop">
    센서 데이터를 OLED 화면에 실시간으로 표시하고,
    웹 대시보드로 그래프를 그리고,
    여러 센서의 데이터를 한눈에 모니터링합니다.
    **데이터가 흐르는 것을 눈으로 보는 순간**, 코딩이 재미있어집니다.
  </Card>

  <Card title="센서 데이터 → 머신러닝" icon="document">
    수집한 데이터로 패턴을 찾고, 예측 모델을 만듭니다.
    "이 소리가 박수인지 말소리인지", "이 움직임이 걷기인지 뛰기인지" —
    센서 데이터를 Teachable Machine이나 Python ML로 학습시킵니다.
    **센서 → 데이터 → 학습 → 예측**, 전체 AI 파이프라인을 경험합니다.
  </Card>
</CardGrid>

## 수업의 흐름

| 단계 | 1막 (1~4차시) | 2막 (5~10차시) | 3막 (11~15차시) |
|------|--------------|----------------|-----------------|
| **코딩 방식** | 직접 코딩 | 바이브 코딩 (AI 활용) | 바이브 코딩 + ML |
| **핵심 활동** | 센서 연결, 기초 코드 | AI에게 코드 시키기, 복합 센서 | 대시보드 제작, 머신러닝 |
| **산출물** | 측정 기록표, 첫 대시보드 | 환경 모니터링 시스템 | ML 모델 + 최종 대시보드 |
| **난이도** | 직접 타이핑 | AI가 생성 → 학생이 수정 | AI가 생성 → 학생이 설계 |
