Grove 커넥터 하나면 충분합니다.
납땜 없이, 꽂기만 하면 센서가 말을 걸기 시작합니다.

교실 온도가 몇 도인지, 복도가 얼마나 시끄러운지, 공기 중 CO₂ 농도는 어떤지 —
센서는 우리가 무심코 지나친 일상을 숫자로 바꿔 줍니다.
그 숫자를 코드로 다루고, AI에게 물어보는 것까지.
이 교재를 마치면, 센서로 데이터를 수집하고 코드로 분석하여 실제 문제를 해결하는 프로젝트를 스스로 설계할 수 있게 됩니다.

---

## 두 가지 코스, 하나의 목표

같은 장비, 같은 센서를 사용하지만 배우는 방식이 다릅니다.
자신에게 맞는 코스를 골라 시작하세요.

<CardGrid>
  <Card title="코스 A: 15차시 수업" icon="open-book">
    **매주 50분, 차근차근 배우기**

    3막 구성으로 이루어진 체계적인 커리큘럼입니다.

    - **1막 — 눈뜨기** (4차시): 센서를 처음 만지고, 첫 코드를 실행합니다
    - **2막 — 파고들기** (6차시): 다양한 센서로 데이터를 모으고 AI에게 분석을 맡깁니다
    - **3막 — 보여주기** (5차시): 팀 프로젝트로 측정하고, 시각화하고, 발표합니다

    센서의 기초부터 팀 프로젝트까지, 15주에 걸쳐 단계별로 배웁니다.

    [코스 A 첫 수업으로 이동하기 →](/code-human-textbook/course-a/act1/lesson-01/)
  </Card>

  <Card title="코스 B: 프로젝트 4레벨" icon="rocket">
    **3시간씩, 레벨업하며 만들기**

    프로젝트를 직접 설계하고 완성하며 실력을 키우는 코스입니다.

    - **Lv.1 탐험가**: 주어진 센서로 첫 프로젝트를 완성합니다
    - **Lv.2 발명가**: 센서 여러 개를 조합해 자기만의 장치를 만듭니다
    - **Lv.3 연구자**: 가설을 세우고, 장기 데이터를 수집해 검증합니다
    - **Lv.4+ 마스터**: 실제 문제를 해결하는 시스템을 설계합니다

    프로젝트 중심으로 배우며, 매번 더 높은 도전에 나섭니다.

    [코스 B 소개 페이지로 이동하기 →](/code-human-textbook/course-b/intro/)
  </Card>
</CardGrid>

## 이 교재의 특징

<CardGrid stagger>
  <Card title="납땜 없는 하드웨어" icon="puzzle">
    Grove 4핀 커넥터로 꽂기만 하면 연결 완료.
    브레드보드도, 점퍼 와이어도, 인두기도 필요 없습니다.
    커넥터에 방향이 정해져 있어서 잘못 꽂을 수도 없습니다.
  </Card>

  <Card title="실생활 데이터" icon="document">
    교실 온도, CO₂ 농도, 소음 레벨, 조도, 심박수까지 —
    교과서 속 숫자가 아니라 **지금 이 공간의 실제 데이터**를 직접 측정합니다.
    측정한 데이터는 파일로 저장하고, 그래프로 그리고, 비교 분석합니다.
  </Card>

  <Card title="AI와 함께" icon="star">
    수집한 데이터를 AI에게 보여주고, 패턴을 찾아달라고 요청합니다.
    "이 데이터에서 뭐가 보여?" 라는 질문 하나로 분석이 시작됩니다.
    AI는 도구이고, 해석은 여러분의 몫입니다.
  </Card>

  <Card title="나만의 프로젝트" icon="laptop">
    아이디어 구상부터 센서 선택, 데이터 수집, 분석, 발표까지 —
    처음부터 끝까지 직접 설계하고 만드는 경험을 합니다.
    정해진 정답이 아니라 **나만의 질문**에서 프로젝트가 시작됩니다.
  </Card>
</CardGrid>
