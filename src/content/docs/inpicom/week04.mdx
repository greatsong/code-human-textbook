#### BME280 읽기 (10-20분)

**[실습 안내 스크립트]**

선생님: "BME280을 읽으려면 드라이버 라이브러리가 필요해요. 먼저 이 파일을 Pico에 저장해야 합니다."

선생님: "Thonny에서 새 파일을 만들어 bme280.py라는 이름으로 Pico에 저장하세요."

:::note[교사 안내: 드라이버 사전 업로드]
BME280 MicroPython 드라이버는 사전에 Pico에 업로드해두는 것을 권장합니다. 수업 중 학생들이 직접 업로드하면 시간이 오래 걸립니다.
:::

**[코드: step1_bme280_read.py]**
```python
# === BME280 기상 데이터 읽기 ===
# 온도, 습도, 기압을 읽어서 출력

from machine import I2C, Pin
import bme280
import time

# --- I2C 설정 ---
i2c = I2C(0, sda=Pin(8), scl=Pin(9))

# --- BME280 센서 초기화 ---
bme = bme280.BME280(i2c=i2c)

# --- 반복 측정 ---
while True:
    temp = bme.values[0]      # 온도 (문자열, 예: "23.45C")
    humidity = bme.values[2]  # 습도 (문자열, 예: "45.2%")
    pressure = bme.values[1]  # 기압 (문자열, 예: "1013.25hPa")

    print(f"온도: {temp}  습도: {humidity}  기압: {pressure}")
    time.sleep(1)
```

선생님: "실행해보세요. 온도, 습도, 기압이 나오죠? 센서에 입김을 불면 온도와 습도가 확 올라갈 거예요."

**[예상 Q&A]**

:::tip[bme280 모듈을 찾을 수 없다고 나와요]
bme280.py 파일이 Pico 안에 저장되어 있어야 해요. Thonny에서 '이 컴퓨터'가 아니라 'MicroPython 장치'에 저장했는지 확인하세요.
:::

:::tip[온도가 실제보다 높게 나와요]
BME280 칩 자체에서 열이 나서 1-2도 높게 나올 수 있어요. 실제 제품에서도 보정값을 적용합니다.
:::

:::note[기압 1013이 뭔 의미예요?]
해수면 기준 표준 대기압이 1013.25hPa이에요. 높은 곳에 가면 기압이 낮아지고, 비가 오려고 하면 기압이 떨어져요. 일기예보에서 '저기압' 하는 게 이거예요.
:::

**[코드: step2_oled_basic.py]**
```python
# === OLED 디스플레이 기본 사용 ===
# 텍스트와 도형을 OLED에 표시

from machine import I2C, Pin
import ssd1306

# --- OLED 설정 ---
i2c = I2C(0, sda=Pin(8), scl=Pin(9))
oled = ssd1306.SSD1306_I2C(128, 64, i2c)  # 128x64 픽셀

# --- 화면 지우기 ---
oled.fill(0)

# --- 텍스트 쓰기 ---
# text(문자열, x좌표, y좌표)
oled.text("Hello Pico!", 0, 0)
oled.text("Weather Station", 0, 16)
oled.text("by Dangok HS", 0, 32)

# --- 화면에 표시! ---
oled.show()  # np.write()처럼 꼭 호출해야 함
```

선생님: "LED에서 np.write()가 필요했듯이, OLED는 oled.show()를 해야 화면에 나와요. 같은 원리!"

#### 실시간 기상 대시보드 (30-40분)

**[코드: step3_weather_station.py]**
```python
# === 미니 기상 스테이션 ===
# BME280 데이터를 OLED에 실시간 표시

from machine import I2C, Pin
import ssd1306
import bme280
import time

# --- I2C 설정 (센서 + OLED 공유) ---
i2c = I2C(0, sda=Pin(8), scl=Pin(9))

# --- 장치 초기화 ---
bme = bme280.BME280(i2c=i2c)
oled = ssd1306.SSD1306_I2C(128, 64, i2c)

# --- 메인 루프 ---
while True:
    temp = bme.values[0]
    pressure = bme.values[1]
    humidity = bme.values[2]

    # 화면 갱신
    oled.fill(0)  # 화면 지우기

    # 제목
    oled.text("= Weather =", 20, 0)

    # 데이터 표시
    oled.text(f"Temp: {temp}", 0, 20)
    oled.text(f"Humi: {humidity}", 0, 35)
    oled.text(f"Pres: {pressure}", 0, 50)

    oled.show()
    time.sleep(1)
```

**[예상 Q&A]**

:::tip[한글은 표시 못 하나요?]
기본 ssd1306 라이브러리는 영문 폰트만 지원해요. 한글 표시는 커스텀 폰트가 필요한데, 메모리를 많이 차지해서 Pico에서는 어려워요. 영문 약어를 쓰는 게 실용적이에요.
:::

:::tip[화면이 깜빡여요]
fill(0) -> 데이터 -> show() 사이에 시간이 걸려서 그래요. 깜빡임이 심하면 sleep 시간을 늘리거나, 변화가 있을 때만 갱신하도록 코드를 수정하면 돼요.
:::

:::tip[OLED 해상도가 128x64인데, 글자 하나가 몇 픽셀이에요?]
기본 폰트는 8x8 픽셀이에요. 그래서 한 줄에 16글자, 총 8줄까지 들어가요.
:::
