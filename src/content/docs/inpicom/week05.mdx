# 스마트 복도 조명 – PIR + LED Strip

## 기본 정보

| 항목 | 내용 |
|------|------|
| 주제 | 모션 감지로 자동 점등하는 스마트 조명 만들기 |
| 시간 | 50분 (1교시) |
| 형태 | 교사 시연 + 모둠 순환 체험 (부품 1세트) |
| 준비물 | Pico 2 WH + Grove Shield, Grove mini PIR 모션 센서, Grove WS2813 LED Strip 1m (60 LED), USB 케이블, AA 배터리홀더 |
| 선수학습 | 2차시 (NeoPixel 제어) |

## 학습목표

1. PIR 모션 센서의 동작 원리(적외선 감지)를 설명할 수 있다
2. 디지털 입력(0/1)을 읽어 LED Strip을 제어할 수 있다
3. 타이머와 상태 변수를 사용하여 자동 소등 기능을 구현할 수 있다

### 실습 1: PIR 센서 이해 (05-20분)

#### 회로 연결 (05-10분)

**[시각 자료: 연결도]**

```
Grove Shield

  ┌─────────────────────────────────┐
  │  [D18]        [D16]            │
  │    ↑            ↑               │
  └────│────────────│───────────────┘
       │            │
  ┌────┴───┐  ┌────┴──────────┐
  │  PIR   │  │  LED Strip    │
  │ 모션   │  │  1m (60 LED)  │
  │ 센서   │  │  IN ────── OUT│
  └────────┘  └───────────────┘

  D18 → GP18 (PIR 디지털 입력)
  D16 → GP16 (LED Strip 데이터)
```

:::note[PIR 센서의 특징]
PIR은 '움직임'을 감지하는 센서이지, '존재'를 감지하는 센서가 아닙니다. 적외선의 **변화**를 읽기 때문에, 가만히 서 있으면 감지하지 못합니다. 이것이 실제 센서등의 한계이기도 합니다.
:::

**[코드: step1_pir_basic.py]**
```python
# === PIR 모션 센서 기본 테스트 ===
# 움직임이 감지되면 "감지!" 출력

from machine import Pin
import time

# --- PIR 센서 설정 ---
# D18 포트 = GP18, 디지털 입력
pir = Pin(18, Pin.IN)

# --- 감지 루프 ---
print("=== 모션 감지 시작 ===")
print("센서 앞에서 움직여보세요!")

count = 0  # 감지 횟수

while True:
    if pir.value() == 1:
        count += 1
        print(f"🚶 감지! (#{count})")
        time.sleep(2)   # PIR은 감지 후 잠시 유지됨
    else:
        print("... 대기 중")
    time.sleep(0.5)
```

선생님: "MQ2는 ADC로 아날로그 값(0~65535)을 읽었죠? PIR은 훨씬 단순해요. 0 아니면 1. 움직임 있으면 1, 없으면 0. 디지털 입력이에요."

**[예상 Q&A]**

Q: "감지 거리가 얼마나 돼요?"
A: "Grove mini PIR은 약 2-3m 정도예요. 각도는 약 100도. 복도 한쪽 벽에 설치하면 지나가는 사람을 충분히 감지해요."

Q: "sleep(2)를 넣은 이유가 뭐예요?"
A: "PIR은 한 번 감지하면 신호가 2-3초간 유지돼요. sleep 없이 읽으면 한 번 움직임에 '감지!' 가 여러 번 출력돼요."

### 실습 2: 스마트 조명 제작 (20-40분)

#### 감지 시 LED 점등 (20-30분)

**[코드: step2_motion_light.py]**
```python
# === 모션 감지 자동 조명 ===
# 사람이 감지되면 LED Strip 점등, 없으면 소등

from machine import Pin
from neopixel import NeoPixel
import time

# --- 하드웨어 설정 ---
pir = Pin(18, Pin.IN)
NUM_LEDS = 60  # LED Strip 1m = 60개
np = NeoPixel(Pin(16), NUM_LEDS)

# --- LED 제어 함수 ---
def lights_on():
    """따뜻한 흰색으로 전체 점등"""
    for i in range(NUM_LEDS):
        np[i] = (255, 200, 100)  # 따뜻한 백색 (전구색)
    np.write()

def lights_off():
    """전체 소등"""
    for i in range(NUM_LEDS):
        np[i] = (0, 0, 0)
    np.write()

# --- 메인 루프 ---
print("=== 스마트 조명 시작 ===")
lights_off()

while True:
    if pir.value() == 1:
        print("🚶 감지 → 점등!")
        lights_on()
        time.sleep(3)  # 최소 3초 유지
    else:
        lights_off()
    time.sleep(0.3)
```

#### 자동 소등 + 부드러운 전환 (30-40분)

**[코드: step3_smart_light.py]**
```python
# === 스마트 조명 v2 – 부드러운 점등/소등 ===
# 페이드 인/아웃 + 자동 소등 타이머

from machine import Pin
from neopixel import NeoPixel
import time

# --- 하드웨어 설정 ---
pir = Pin(18, Pin.IN)
NUM_LEDS = 60
np = NeoPixel(Pin(16), NUM_LEDS)

# --- 설정값 ---
AUTO_OFF_SEC = 10   # 감지 없으면 10초 후 소등
WARM_WHITE = (255, 200, 100)  # 전구색

# --- 부드러운 전환 함수 ---
def fade_in(steps=20):
    """서서히 밝아지기"""
    for s in range(1, steps + 1):
        brightness = s / steps
        r = int(WARM_WHITE[0] * brightness)
        g = int(WARM_WHITE[1] * brightness)
        b = int(WARM_WHITE[2] * brightness)
        for i in range(NUM_LEDS):
            np[i] = (r, g, b)
        np.write()
        time.sleep(0.03)

def fade_out(steps=20):
    """서서히 어두워지기"""
    for s in range(steps, 0, -1):
        brightness = s / steps
        r = int(WARM_WHITE[0] * brightness)
        g = int(WARM_WHITE[1] * brightness)
        b = int(WARM_WHITE[2] * brightness)
        for i in range(NUM_LEDS):
            np[i] = (r, g, b)
        np.write()
        time.sleep(0.03)
    # 완전 소등
    for i in range(NUM_LEDS):
        np[i] = (0, 0, 0)
    np.write()

# --- 메인 루프 ---
print("=== 스마트 조명 v2 ===")
is_on = False
last_detect = time.ticks_ms()

while True:
    if pir.value() == 1:
        last_detect = time.ticks_ms()  # 감지 시각 갱신
        if not is_on:
            print("🚶 감지 → 페이드 인!")
            fade_in()
            is_on = True

    # 자동 소등: 마지막 감지 후 N초 경과
    elapsed = time.ticks_diff(time.ticks_ms(), last_detect)
    if is_on and elapsed > AUTO_OFF_SEC * 1000:
        print("⏰ 타임아웃 → 페이드 아웃")
        fade_out()
        is_on = False

    time.sleep(0.2)
```

선생님: "v2에서는 확 켜지고 확 꺼지는 대신, 서서히 밝아지고 서서히 어두워져요. 실제 고급 센서등에 적용되는 효과예요."

**[예상 Q&A]**

Q: "time.ticks_ms()가 뭐예요?"
A: "Pico가 시작된 이후 흘러간 시간을 밀리초로 알려줘요. ticks_diff로 두 시점의 차이를 구해서 '얼마나 시간이 지났나' 계산하는 거예요."

Q: "60개 LED에 전력이 충분한가요?"
A: "USB로는 부족할 수 있어요. 60개 전부 흰색(255,255,255)이면 최대 3.6A가 필요한데 USB는 0.5A 정도. 밝기를 낮추거나 배터리팩을 따로 연결해야 해요. 오늘 코드에서 (255,200,100)으로 한 이유이기도 해요."

:::caution[LED Strip 전력 주의]
60개 LED를 최대 밝기(255,255,255)로 켜면 약 3.6A가 필요합니다. USB 전원(0.5A)으로는 부족하므로, 밝기를 낮추거나 별도 배터리팩을 연결하세요.
:::

Q: "AUTO_OFF_SEC를 바꿀 수 있나요?"
A: "당연하죠! 실제 복도등은 보통 30초~1분으로 설정해요. 장소와 용도에 따라 조절하는 거예요."
