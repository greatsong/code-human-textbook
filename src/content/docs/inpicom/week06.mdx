### 실습 1: 음성인식 센서 기초 (05-20분)

#### 회로 연결 + UART (05-12분)

**[시각 자료]**

```
Grove Shield

  ┌─────────────────────────────────┐
  │  [UART]       [D16]            │
  │    ↑            ↑               │
  └────│────────────│───────────────┘
       │            │
  ┌────┴──────┐ ┌──┴───────────┐
  │ Speech    │ │ RGB LED      │
  │ Recognizer│ │ Stick        │
  └───────────┘ └──────────────┘

  UART → GP0(TX), GP1(RX)
  D16  → GP16 (NeoPixel)
```

:::tip[UART 교차 연결]
UART는 TX(송신)와 RX(수신)를 **교차 연결**합니다. 센서의 TX가 Pico의 RX로, 센서의 RX가 Pico의 TX로 연결됩니다. 전화기처럼 말하는 쪽(TX)과 듣는 쪽(RX)이 서로 연결되는 것입니다.
:::

#### 인식 결과 확인 (12-20분)

**[코드: step1_speech_test.py]**
```python
# === 음성인식 센서 테스트 ===
# 센서가 인식한 명령어 번호를 확인

from machine import UART, Pin
import time

# --- UART 설정 ---
# Grove UART 포트: TX=GP0, RX=GP1
uart = UART(0, baudrate=9600, tx=Pin(0), rx=Pin(1))

# --- 내장 명령어 목록 (Grove Speech Recognizer 기본) ---
# 센서마다 다를 수 있으니 데이터시트 확인!
commands = {
    1: "Turn on the light",
    2: "Turn off the light",
    3: "Play music",
    4: "Pause",
    5: "Next",
    6: "Previous",
    7: "Up (volume up)",
    8: "Down (volume down)",
    9: "Turn on AC",
    10: "Turn off AC",
}

print("=== 음성인식 테스트 ===")
print("센서에 대고 영어로 말해보세요!")
print("예: 'Turn on the light'")

while True:
    if uart.any():
        data = uart.read()
        if data:
            cmd_id = data[0]  # 첫 바이트가 명령어 번호
            cmd_name = commands.get(cmd_id, f"알 수 없는 명령({cmd_id})")
            print(f"인식됨: #{cmd_id} = {cmd_name}")
    time.sleep(0.1)
```

선생님: "이 센서는 영어 명령어가 기본 내장돼 있어요. 'Turn on the light'이라고 말해보세요."

:::caution[교사 안내]
Grove Speech Recognizer [101020232]는 내장 영어 명령어를 사용합니다. 조용한 환경에서 센서에 가까이(10cm 이내) 대고 명확하게 발음해야 합니다. 센서 모델에 따라 명령어 세트와 프로토콜이 다를 수 있으니, 반드시 해당 센서의 데이터시트를 확인하고 코드를 조정하세요.
:::

**[코드: step2_voice_led.py]**
```python
# === 음성 제어 무드등 ===
# 음성 명령에 따라 LED 색상 변경

from machine import UART, Pin
from neopixel import NeoPixel
import time

# --- 하드웨어 설정 ---
uart = UART(0, baudrate=9600, tx=Pin(0), rx=Pin(1))
NUM_LEDS = 10
np = NeoPixel(Pin(16), NUM_LEDS)

# --- LED 제어 ---
def set_all(r, g, b):
    for i in range(NUM_LEDS):
        np[i] = (r, g, b)
    np.write()

# --- 명령어 → LED 동작 매핑 ---
# 센서 내장 명령어를 LED 색상에 연결!
def handle_command(cmd_id):
    """명령어 번호에 따라 LED 색상 변경"""
    if cmd_id == 1:   # "Turn on the light"
        set_all(255, 255, 200)
        print("💡 조명 ON (따뜻한 백색)")

    elif cmd_id == 2: # "Turn off the light"
        set_all(0, 0, 0)
        print("⚫ 조명 OFF")

    elif cmd_id == 3: # "Play music"
        set_all(0, 100, 255)
        print("🔵 뮤직 모드 (파랑)")

    elif cmd_id == 4: # "Pause"
        set_all(255, 100, 0)
        print("🟠 휴식 모드 (주황)")

    elif cmd_id == 7: # "Up" (volume up)
        # 현재 밝기 올리기
        set_all(255, 255, 255)
        print("🔆 밝기 UP")

    elif cmd_id == 8: # "Down" (volume down)
        set_all(30, 30, 30)
        print("🔅 밝기 DOWN")

    elif cmd_id == 5: # "Next"
        set_all(255, 0, 100)
        print("🩷 다음 색상 (핑크)")

    elif cmd_id == 6: # "Previous"
        set_all(0, 255, 100)
        print("🟢 이전 색상 (민트)")

    else:
        print(f"미등록 명령: {cmd_id}")

# --- 메인 루프 ---
print("=== 음성 제어 무드등 ===")
print("영어로 명령해보세요!")
set_all(0, 0, 0)  # 시작 시 꺼짐

while True:
    if uart.any():
        data = uart.read()
        if data:
            handle_command(data[0])
    time.sleep(0.1)
```

#### 패턴 명령 추가 (30-40분)

**[코드: step3_voice_pattern.py]**
```python
# === 음성 제어 무드등 v2 – 패턴 추가 ===
# 기본 색상 + 무지개/숨쉬기 애니메이션 패턴

from machine import UART, Pin
from neopixel import NeoPixel
import time

uart = UART(0, baudrate=9600, tx=Pin(0), rx=Pin(1))
NUM_LEDS = 10
np = NeoPixel(Pin(16), NUM_LEDS)

# --- LED 기본 함수 ---
def set_all(r, g, b):
    for i in range(NUM_LEDS):
        np[i] = (r, g, b)
    np.write()

def wheel(pos):
    if pos < 85:
        return (255 - pos * 3, pos * 3, 0)
    elif pos < 170:
        pos -= 85
        return (0, 255 - pos * 3, pos * 3)
    else:
        pos -= 170
        return (pos * 3, 0, 255 - pos * 3)

# --- 애니메이션 패턴 (2차시 복습!) ---
def rainbow_loop():
    """무지개 1바퀴"""
    for offset in range(256):
        for i in range(NUM_LEDS):
            np[i] = wheel((i * 256 // NUM_LEDS + offset) % 256)
        np.write()
        time.sleep(0.02)

def breathe_loop(r, g, b):
    """숨쉬기 1사이클"""
    for brightness in range(0, 100, 5):
        s = brightness / 100
        set_all(int(r*s), int(g*s), int(b*s))
        time.sleep(0.02)
    for brightness in range(100, 0, -5):
        s = brightness / 100
        set_all(int(r*s), int(g*s), int(b*s))
        time.sleep(0.02)

# --- 상태 관리 ---
current_mode = "off"  # off, static, rainbow, breathe

def handle(cmd_id):
    global current_mode

    if cmd_id == 1:    # "Turn on the light"
        set_all(255, 200, 100)
        current_mode = "static"
        print("💡 조명 ON")

    elif cmd_id == 2:  # "Turn off the light"
        set_all(0, 0, 0)
        current_mode = "off"
        print("⚫ OFF")

    elif cmd_id == 3:  # "Play music" → 무지개 모드!
        current_mode = "rainbow"
        print("🌈 무지개 모드!")

    elif cmd_id == 4:  # "Pause" → 숨쉬기 모드
        current_mode = "breathe"
        print("😮‍💨 숨쉬기 모드")

    elif cmd_id == 5:  # "Next" → 색상 순환
        colors = [(255,0,0),(0,255,0),(0,0,255),
                  (255,255,0),(255,0,255),(0,255,255)]
        import random
        c = random.choice(colors)
        set_all(*c)
        current_mode = "static"
        print(f"🎲 랜덤 색상: {c}")

# --- 메인 루프 ---
print("=== 음성 제어 무드등 v2 ===")
set_all(0, 0, 0)

while True:
    # 음성 명령 확인
    if uart.any():
        data = uart.read()
        if data:
            handle(data[0])

    # 현재 모드에 따라 애니메이션 실행
    if current_mode == "rainbow":
        rainbow_loop()
    elif current_mode == "breathe":
        breathe_loop(0, 100, 255)
    else:
        time.sleep(0.1)
```

선생님: "이제 'Play music'이라고 하면 무지개가 흘러가고, 'Pause'하면 숨쉬기 모드, 'Turn off the light'하면 꺼져요. 2차시에서 배운 패턴이 여기서 다시 쓰이죠!"

**[예상 Q&A]**

Q: "global이 뭐예요?"
A: "함수 안에서 함수 밖의 변수를 바꾸고 싶을 때 써요. global current_mode라고 선언하면 handle() 함수 안에서 current_mode를 바꿀 수 있어요."

Q: "무지개 도중에 명령이 안 먹어요"
A: "rainbow_loop()가 256번 반복하는 동안 uart.any()를 체크하지 않아서 그래요. 더 정교하게 만들려면 rainbow 안에서도 uart를 체크해야 해요. 이건 '비동기 처리'라는 개념인데, 고급 주제예요."

Q: "random이 import 안 돼요"
A: "MicroPython에서는 import random이 아니라 from random import choice 형태로 써야 할 수 있어요. 안 되면 colors[0]처럼 고정해도 괜찮아요."

오늘 배운 세 가지 통신 정리:

┌────────────┬──────────────────────────┐
│ 통신 방식   │ 특징                     │
├────────────┼──────────────────────────┤

:::note[음성인식의 두 가지 방식]
- **로컬 (오늘)**: 10개 명령어, 인터넷 불필요, 빠름
- **클라우드 (시리)**: 무한 명령어, 인터넷 필수, 느림

IoT 제품 설계 시 '어디서 처리할 것인가'가 핵심 선택입니다. 이것이 **Edge AI vs Cloud AI**!
:::
