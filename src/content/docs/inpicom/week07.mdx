# CO2 환기 알리미 – SCD30 + LED Stick + OLED

## 기본 정보

| 항목 | 내용 |
|------|------|
| 주제 | CO2 농도 측정 + LED 경보 + OLED 수치 표시 |
| 시간 | 50분 (1교시) |
| 형태 | 교사 시연 + 모둠 순환 체험 (부품 1세트) |
| 준비물 | Pico 2 WH + Grove Shield, Grove SCD30 (CO2+온습도), Grove RGB LED Stick, Grove OLED 0.96", USB 케이블 |

## 학습목표

1. SCD30 센서로 CO2 농도를 측정하고 ppm 단위를 이해할 수 있다
2. CO2 농도에 따른 환기 판단 기준을 설정할 수 있다
3. OLED + LED 이중 출력으로 실용적인 모니터링 시스템을 만들 수 있다

:::caution[교사 안내]
SCD30은 I2C 주소 0x61입니다. `scd30.py` 드라이버를 Pico에 사전 업로드하세요. SCD30은 측정 주기가 2초이므로 `sleep(2)` 이상 필요합니다.
:::

**[코드: step1_co2_read.py]**
```python
# === SCD30 CO2 센서 읽기 ===
# CO2 농도(ppm), 온도, 습도 측정

from machine import I2C, Pin
import scd30
import time

# --- I2C 설정 ---
i2c = I2C(0, sda=Pin(8), scl=Pin(9))

# --- SCD30 초기화 ---
sensor = scd30.SCD30(i2c, addr=0x61)
sensor.start_continous_measurement()

print("=== CO2 모니터 시작 ===")
print("SCD30 예열 중... (최초 몇 회 값이 불안정할 수 있음)")

while True:
    if sensor.data_ready():
        co2, temp, humidity = sensor.read_measurement()
        print(f"CO2: {co2:.0f} ppm | 온도: {temp:.1f}°C | 습도: {humidity:.1f}%")
    time.sleep(2)  # SCD30 측정 주기 = 2초
```

**[코드: step2_co2_led.py]**
```python
# === CO2 환기 알리미 – LED 경보 ===
# CO2 농도에 따라 LED 색상 변경

from machine import I2C, Pin
from neopixel import NeoPixel
import scd30
import time

# --- 하드웨어 ---
i2c = I2C(0, sda=Pin(8), scl=Pin(9))
sensor = scd30.SCD30(i2c, addr=0x61)
sensor.start_continous_measurement()

NUM_LEDS = 10
np = NeoPixel(Pin(16), NUM_LEDS)

# --- CO2 임계값 (ppm) ---
GOOD = 800       # 양호
WARN = 1000      # 주의
BAD  = 1500      # 환기 필요

def set_all(r, g, b):
    for i in range(NUM_LEDS):
        np[i] = (r, g, b)
    np.write()

# --- 메인 루프 ---
print("=== CO2 환기 알리미 ===")

while True:
    if sensor.data_ready():
        co2, temp, hum = sensor.read_measurement()

        if co2 < GOOD:
            set_all(0, 50, 0)       # 🟢 양호
            status = "양호"
        elif co2 < WARN:
            set_all(0, 100, 0)      # 🟢 좋음 (밝은 초록)
            status = "좋음"
        elif co2 < BAD:
            set_all(255, 200, 0)    # 🟡 주의
            status = "주의"
        else:
            set_all(255, 0, 0)      # 🔴 환기 필요!
            status = "환기!"

        print(f"CO2: {co2:.0f}ppm → {status}")

    time.sleep(2)
```

#### OLED 대시보드 통합 (30-40분)

**[코드: step3_co2_dashboard.py]**
```python
# === CO2 환기 알리미 – 풀 대시보드 ===
# OLED에 수치 + LED에 색상 경보

from machine import I2C, Pin
from neopixel import NeoPixel
import ssd1306
import scd30
import time

# --- 하드웨어 ---
i2c = I2C(0, sda=Pin(8), scl=Pin(9))
sensor = scd30.SCD30(i2c, addr=0x61)
sensor.start_continous_measurement()
oled = ssd1306.SSD1306_I2C(128, 64, i2c)
NUM_LEDS = 10
np = NeoPixel(Pin(16), NUM_LEDS)

# --- CO2 임계값 ---
WARN = 1000
BAD  = 1500

def set_all(r, g, b):
    for i in range(NUM_LEDS):
        np[i] = (r, g, b)
    np.write()

def get_status(co2):
    """CO2 값에 따른 상태 판정"""
    if co2 < WARN:
        set_all(0, 50, 0)
        return "GOOD"
    elif co2 < BAD:
        set_all(255, 200, 0)
        return "WARN - Ventilate?"
    else:
        set_all(255, 0, 0)
        return "BAD - Open Window!"

def draw_bar(oled, co2):
    """CO2 농도를 막대 그래프로 표시"""
    # 막대 너비 계산 (0~2000ppm → 0~120px)
    bar_width = min(int(co2 / 2000 * 120), 120)
    oled.rect(4, 50, 120, 10, 1)         # 테두리
    oled.fill_rect(4, 50, bar_width, 10, 1)  # 채움

# --- 메인 루프 ---
print("=== CO2 대시보드 시작 ===")

while True:
    if sensor.data_ready():
        co2, temp, hum = sensor.read_measurement()
        status = get_status(co2)

        # OLED 갱신
        oled.fill(0)
        oled.text("CO2 Monitor", 20, 0)
        oled.text(f"CO2: {co2:.0f} ppm", 0, 14)
        oled.text(f"T:{temp:.1f}C H:{hum:.0f}%", 0, 28)
        oled.text(status, 0, 40)
        draw_bar(oled, co2)
        oled.show()

        print(f"CO2: {co2:.0f}ppm | {status}")

    time.sleep(2)
```

**[예상 Q&A]**

Q: "교실에서 실제로 1500 넘는 경우가 있나요?"
A: "자주 있어요! 30명이 1시간 동안 창문 닫고 수업하면 2000ppm 넘는 경우도 있어요. 실제로 이 센서로 측정해보면 깜짝 놀랄 거예요."

Q: "CO2가 높으면 왜 졸려요?"
A: "CO2 자체가 독성이 있는 건 아니에요. 하지만 CO2가 높다는 건 환기가 안 되고 있다는 뜻이고, 상대적으로 산소 비율이 낮아져서 뇌에 산소 공급이 줄어 졸리는 거예요."

Q: "SCD30은 MQ2보다 왜 비싸요?"
A: "측정 원리가 다릅니다. MQ2는 금속산화물의 저항 변화를 보는 저가형이고, SCD30은 NDIR(비분산적외선) 방식으로 CO2 분자가 적외선을 흡수하는 양을 측정해요. 훨씬 정확하지만 부품이 복잡해요."

:::tip[SCD30 vs MQ2 비교]
- **MQ2**: 금속산화물 저항 변화, 저가형, 정성적 측정 ("가스가 있다/없다")
- **SCD30**: NDIR(비분산적외선) 방식, 고가형, 정량적 측정 ("CO2가 1234ppm이다")

정확한 수치가 필요한 환경 모니터링에는 SCD30 같은 고정밀 센서가 필수입니다.
:::
