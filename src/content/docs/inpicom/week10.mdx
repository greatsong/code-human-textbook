### 실습 1: 기압 -> 고도 변환 (05-20분)

<Mermaid client:load chart={`

`} caption="보정(calibration) 개념: 현재 위치를 기준점(0m)으로 설정한 뒤 상대 고도를 계산" />

**[코드: step1_altitude.py]**
```python
# === 기압으로 고도 계산 ===
# 바로미터 공식으로 상대 고도 추정

from machine import I2C, Pin
import bme280
import time

i2c = I2C(0, sda=Pin(8), scl=Pin(9))
bme = bme280.BME280(i2c=i2c)

# --- 고도 계산 함수 ---
def calc_altitude(pressure_hpa, sea_level=1013.25):
    """기압(hPa)으로 고도(m) 추정
    국제 기압-고도 공식 (간략 버전)"""
    return 44330.0 * (1.0 - (pressure_hpa / sea_level) ** 0.1903)

# --- 기준점 보정 ---
# 현재 위치의 기압을 기준(0m)으로 설정
print("기준점 보정 중... 5초간 가만히 계세요.")
total = 0
for i in range(10):
    p_str = bme.values[1]  # "1013.25hPa"
    p = float(p_str[:-3])  # 숫자만 추출
    total += p
    time.sleep(0.5)
base_pressure = total / 10
base_altitude = calc_altitude(base_pressure)
print(f"기준 기압: {base_pressure:.2f} hPa")
print(f"기준 고도: {base_altitude:.1f} m")

# --- 실시간 측정 ---
print("\n=== 고도 측정 시작 ===")
print("계단을 오르내려 보세요!")

while True:
    p_str = bme.values[1]
    pressure = float(p_str[:-3])
    altitude = calc_altitude(pressure)
    relative = altitude - base_altitude  # 기준점 대비 상대 고도

    floor = relative / 3.0  # 1층 ≈ 3m로 가정

    print(f"기압: {pressure:.1f}hPa | 상대고도: {relative:+.1f}m | 약 {floor:+.1f}층")
    time.sleep(1)
```

선생님: "핵심은 '기준점 보정'이에요. 시작할 때 현재 기압을 측정해서 0m로 놓는 거예요. 그 다음부터는 기압 변화만으로 몇 미터 올라갔는지 계산해요."

**[예상 Q&A]**

:::tip[44330이라는 숫자는 어디서 나온 거예요?]
국제 기압-고도 공식에서 나온 상수예요. 대기의 온도와 밀도를 고려해서 유도된 값이에요. 외울 필요 없고, 이런 공식이 있다는 것만 알면 돼요.
:::

:::note[1층이 정확히 3m인가요?]
건물마다 달라요. 일반 건물은 2.7~3.3m, 상가는 4~5m. 우리 학교 층고를 직접 재서 넣으면 더 정확해져요.
:::

:::caution[날씨가 변하면 기압도 변합니다]
맞아요! 그래서 시작할 때 보정하는 거예요. 장시간 사용하면 날씨 변화로 오차가 생겨요. 전문 장비는 GPS와 함께 써서 보정합니다.
:::

### 실습 2: 고도 시각화 (20-40분)

#### OLED 고도계 (20-30분)

**[코드: step2_altimeter_oled.py]**
```python
# === OLED 고도계 ===
# 기압, 상대 고도, 층수를 OLED에 표시

from machine import I2C, Pin
import ssd1306
import bme280
import time

i2c = I2C(0, sda=Pin(8), scl=Pin(9))
bme = bme280.BME280(i2c=i2c)
oled = ssd1306.SSD1306_I2C(128, 64, i2c)

def calc_altitude(p, sea=1013.25):
    return 44330.0 * (1.0 - (p / sea) ** 0.1903)

# --- 기준점 보정 ---
oled.fill(0)
oled.text("Calibrating...", 0, 28)
oled.show()

total = 0
for _ in range(10):
    total += float(bme.values[1][:-3])
    time.sleep(0.5)
base_p = total / 10
base_alt = calc_altitude(base_p)

# --- 메인 ---
while True:
    p = float(bme.values[1][:-3])
    t = bme.values[0]
    alt = calc_altitude(p) - base_alt
    floor_num = alt / 3.0

    oled.fill(0)
    oled.text("= Altimeter =", 15, 0)
    oled.text(f"Press: {p:.1f}hPa", 0, 16)
    oled.text(f"Alt: {alt:+.1f} m", 0, 30)
    oled.text(f"Floor: {floor_num:+.1f} F", 0, 44)
    oled.text(f"Temp: {t}", 0, 56)
    oled.show()

    time.sleep(1)
```

**[코드: step3_floor_indicator.py]**
```python
# === LED 층수 표시기 ===
# 층수에 따라 LED가 하나씩 켜짐 (계단 실험용)

from machine import I2C, Pin
from neopixel import NeoPixel
import ssd1306
import bme280
import time

i2c = I2C(0, sda=Pin(8), scl=Pin(9))
bme = bme280.BME280(i2c=i2c)
oled = ssd1306.SSD1306_I2C(128, 64, i2c)
NUM_LEDS = 10
np = NeoPixel(Pin(16), NUM_LEDS)

def calc_altitude(p, sea=1013.25):
    return 44330.0 * (1.0 - (p / sea) ** 0.1903)

# --- 기준점 보정 ---
total = 0
for _ in range(10):
    total += float(bme.values[1][:-3])
    time.sleep(0.5)
base_alt = calc_altitude(total / 10)

def update_floor_led(floor_num):
    """층수에 따라 LED 표시
    0층=LED 0 (파랑), 5층=LED 5 (초록), 지하=꺼짐"""
    for i in range(NUM_LEDS):
        np[i] = (0, 0, 0)

    # 현재 층에 해당하는 LED 켜기
    led_idx = max(0, min(9, int(floor_num + 0.5)))

    # 높이에 따라 파랑→초록→빨강
    if led_idx < 4:
        np[led_idx] = (0, 0, 200)     # 저층: 파랑
    elif led_idx < 7:
        np[led_idx] = (0, 200, 0)     # 중층: 초록
    else:
        np[led_idx] = (200, 0, 0)     # 고층: 빨강

    # 이전 층까지 흐릿하게 표시 (궤적)
    for j in range(led_idx):
        np[j] = (10, 10, 10)

    np.write()

# --- 메인 ---
print("=== 층수 표시기 ===")
print("배터리 연결 후 계단을 오르내려 보세요!")

while True:
    p = float(bme.values[1][:-3])
    alt = calc_altitude(p) - base_alt
    floor_num = alt / 3.0

    update_floor_led(floor_num)

    # OLED 표시
    oled.fill(0)
    oled.text(f"Floor: {floor_num:+.1f}", 0, 0)
    oled.text(f"Alt: {alt:+.1f}m", 0, 16)

    # 간이 건물 그림
    for f in range(5):
        y = 60 - f * 10
        oled.rect(80, y, 40, 10, 1)
        if int(floor_num + 0.5) == f:
            oled.fill_rect(82, y + 2, 36, 6, 1)

    oled.show()
    time.sleep(0.5)
```

### 정리 (40-50분)

```
같은 센서(BME280), 다른 활용:

4차시: 온도·습도·기압 → OLED 기상 스테이션
9차시: 온도 → LED Strip 색상 시각화
10차시: 기압 → 고도 추정 + 층수 표시

하나의 센서에서 어떤 데이터를 '어떻게 해석하느냐'가
프로젝트의 방향을 결정합니다.

이것이 '문제 정의'의 힘!
```

:::caution[핵심 정리]
같은 BME280 센서라도 온도 데이터를 사용하면 시각화 조명(9차시), 기압 데이터를 사용하면 고도계(10차시)가 됩니다. **데이터를 어떻게 해석하느냐**가 프로젝트의 방향을 결정합니다.
:::
