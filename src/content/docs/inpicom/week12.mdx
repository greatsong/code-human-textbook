:::tip[이중 검증의 핵심]
MQ2(가스)와 SCD30(CO2) 두 센서가 **동시에** 임계값을 넘어야 검출로 판정합니다. 이렇게 하면 손소독제(MQ2만 반응)나 사람이 많은 곳(CO2만 상승) 같은 오탐을 크게 줄일 수 있어요.
:::

### 실습 1: 이중 검증 설계 (05-20분)

**[코드: step1_dual_sensor.py]**
```python
# === MQ2 + SCD30 동시 읽기 ===
# 두 센서의 반응 패턴 비교

from machine import I2C, Pin, ADC
import scd30
import time

# --- 하드웨어 ---
i2c = I2C(0, sda=Pin(8), scl=Pin(9))
gas = ADC(Pin(26))  # MQ2 (ADC)
co2_sensor = scd30.SCD30(i2c, addr=0x61)  # SCD30 (I2C)
co2_sensor.start_continous_measurement()

def read_mq2_avg(n=10):
    total = 0
    for _ in range(n):
        total += gas.read_u16()
        time.sleep_ms(10)
    return total // n

print("=== 이중 센서 모니터 ===")
print("MQ2(ADC)와 SCD30(I2C)를 동시에 관찰합니다")
print("손소독제 vs 입김의 차이를 비교해보세요!")
time.sleep(3)

while True:
    mq2_val = read_mq2_avg()

    if co2_sensor.data_ready():
        co2, temp, hum = co2_sensor.read_measurement()
        print(f"MQ2: {mq2_val:5d} | CO2: {co2:6.0f}ppm | T:{temp:.1f}C")

    time.sleep(2)
```

**[코드: step2_dual_detector.py]**
```python
# === 전자담배 검출기 풀버전 ===
# MQ2 + SCD30 이중 검증 + OLED + LED

from machine import I2C, Pin, ADC
from neopixel import NeoPixel
import ssd1306
import scd30
import time

# --- 하드웨어 ---
i2c = I2C(0, sda=Pin(8), scl=Pin(9))
gas = ADC(Pin(26))
co2_sensor = scd30.SCD30(i2c, addr=0x61)
co2_sensor.start_continous_measurement()
oled = ssd1306.SSD1306_I2C(128, 64, i2c)
NUM_LEDS = 10
np = NeoPixel(Pin(16), NUM_LEDS)

# --- 임계값 ---
MQ2_THRESHOLD = 25000
CO2_BASELINE = 600     # 평상시 CO2
CO2_SPIKE = 200        # 기준 대비 이 이상 상승하면 의심

# --- 상태 ---
co2_baseline = CO2_BASELINE  # 실행 시 자동 보정됨
detection_count = 0
log = []  # 검출 로그

def read_mq2_avg(n=10):
    total = 0
    for _ in range(n):
        total += gas.read_u16()
        time.sleep_ms(10)
    return total // n

def set_leds(r, g, b):
    for i in range(NUM_LEDS):
        np[i] = (r, g, b)
    np.write()

def blink_alert():
    for _ in range(3):
        set_leds(255, 0, 0)
        time.sleep(0.1)
        set_leds(0, 0, 0)
        time.sleep(0.1)

# --- 판정 로직 ---
def judge(mq2, co2):
    """이중 검증으로 판정
    둘 다 높으면 = 검출 (전자담배 가능성 높음)
    MQ2만 높으면 = 화학물질 (손소독제 등)
    CO2만 높으면 = 환기 필요 (사람 많음)
    둘 다 정상이면 = 안전"""
    global detection_count, log

    mq2_high = mq2 > MQ2_THRESHOLD
    co2_high = co2 > (co2_baseline + CO2_SPIKE)

    if mq2_high and co2_high:
        detection_count += 1
        timestamp = time.ticks_ms() // 1000
        log.append(f"{timestamp}s: MQ2={mq2} CO2={co2:.0f}")
        if len(log) > 10:
            log.pop(0)  # 최근 10건만 유지
        return "DETECTED", (255, 0, 0)
    elif mq2_high:
        return "CHEMICAL", (255, 165, 0)
    elif co2_high:
        return "VENTILATE", (255, 255, 0)
    else:
        return "SAFE", (0, 50, 0)

# --- 기준값 보정 ---
oled.fill(0)
oled.text("Calibrating...", 0, 28)
oled.show()
print("기준값 보정 중...")
time.sleep(5)

total_co2 = 0
count = 0
for _ in range(5):
    if co2_sensor.data_ready():
        c, _, _ = co2_sensor.read_measurement()
        total_co2 += c
        count += 1
    time.sleep(2)
if count > 0:
    co2_baseline = total_co2 / count
print(f"CO2 기준값: {co2_baseline:.0f}ppm")

# --- 메인 루프 ---
print("=== 전자담배 검출기 풀버전 가동 ===")

while True:
    mq2_val = read_mq2_avg()

    if co2_sensor.data_ready():
        co2, temp, hum = co2_sensor.read_measurement()

        status, color = judge(mq2_val, co2)

        # LED
        if status == "DETECTED":
            blink_alert()
        else:
            set_leds(*color)

        # OLED
        oled.fill(0)
        oled.text(f"[{status}]", 0, 0)
        oled.text(f"MQ2: {mq2_val}", 0, 14)
        oled.text(f"CO2: {co2:.0f}ppm", 0, 26)
        oled.text(f"T:{temp:.1f}C H:{hum:.0f}%", 0, 38)
        oled.text(f"Detect: {detection_count}", 0, 52)
        oled.show()

        print(f"[{status}] MQ2:{mq2_val} CO2:{co2:.0f} 검출:{detection_count}회")

    time.sleep(2)
```

**[예상 Q&A]**

:::note[Q: "실제 전자담배가 없는데 어떻게 테스트해요?"]
입김 + 손소독제를 동시에 사용하면 비슷한 효과를 낼 수 있어요. 또는 라이터 가스(부탄)를 살짝 근처에서 틀면 MQ2가 반응해요. 안전에 주의하세요!
:::

:::note[Q: "log에 저장한 건 전원 끄면 사라지나요?"]
네, 메모리에만 있으니까요. 영구 저장하려면 Pico의 파일 시스템에 기록하거나, Wi-Fi로 서버에 전송해야 해요.
